# Cursor Marketing Page Clone (HTML & CSS)

This project is a **desktop-only clone of the Cursor marketing website**, built using **only HTML and CSS**.  
The goal was to closely recreate the layout, visual hierarchy, typography, and color scheme of the original site while strictly following the given constraints.
